# Program Usage

This program is designed to perform network programming tasks in the context of the "idobata" project at the university.

## Installation

1. Clone the repository: `git clone https://github.com/your-username/idobata.git`
2. Navigate to the project directory: `cd idobata`
3. Install the required dependencies: `npm install`
